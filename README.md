# Arduino-Compass:
A compass with arrow always point to the North and print q as the Qiblah direction. When facing the Qiblah the led of arduino will flicker.

## Materials used:
* Arduino mega
* LSM303 Accelemetor Sensor
* Nokia 5110 Screen
## Connections:
#### Screen pins:
  * rst     6
  * ce      7
  * dc      5
  * din     4
  * clk     8
  * vcc     3.3
  * light   11
  * gnd     gnd
#### LSM303:
  * scl   scl
  * sda   sda
  * gnd   gnd
  * vcc   vcc
